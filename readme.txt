This is the Coleman MAGMA library v1.3 for computing Coleman
integrals on general curves over Q. For information on how to
use the code see examples.pdf.

April 2018

Jan Tuitman (joint work with Jennifer Balakrishnan)
jan.tuitman@kuleuven.

----
Additional functionality is provided in coleman_extensions.m. For information
on how to use the new functionality (like integrating to points over
number fields or executing Chabauty-Coleman for a wider class of curves),
see ColemanExtensions.pdf. 

Joint work with Sachi Hashimoto.
